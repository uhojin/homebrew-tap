# Homebrew Tap

Homebrew formulae for uhojin's projects.

## Usage

```
brew tap uhojin/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `mercury` | Tiny macOS temperature monitor for your menu bar |

```
brew install --cask mercury
```

Or install directly without tapping first:

```
brew install --cask uhojin/tap/mercury
```
